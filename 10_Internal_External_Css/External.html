External CSS Example

What is External CSS?

External CSS is a method of styling where CSS rules are written in a separate file with a .css extension. This file is then linked to the HTML document using the <link> tag in the <head> section.

Benefits of External CSS

External CSS provides better organization, reusability, and maintainability. It allows you to style multiple pages using a single CSS file, making it easier to maintain consistent styling across your website.
